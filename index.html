★★★★★

Live-detection settings

Without an API key, "Now Playing" and the Live Now tab can't confirm what's actually live — they'll only let you jump to a channel and check yourself. Adding a free YouTube Data API key lets the app genuinely search YouTube for live plane-spotting streams — both airport-specific and worldwide — ranked by live viewer count, with real thumbnails. It re-checks automatically every 60 minutes; YouTube caps live-search calls at roughly 100/day per key, and each refresh uses about 4, so this cadence leaves plenty of room for manual refreshes too.

Stored privately in this browser only — not shared with anyone else using this app, and never leaves your device except to call YouTube's API directly.
